# 贴吧自动签到网站

一个多用户、服务器端执行的百度贴吧签到控制台。用户手动填写 `BDUSS`，系统自动获取关注贴吧，用户确认后再执行签到。

## 功能

- 用户注册、登录、退出
- 用户修改登录密码
- 第一个注册用户自动成为管理员
- 手动添加百度账号 `BDUSS`
- `BDUSS` 使用 `AES-GCM` 加密存储
- 自动同步账号关注的贴吧
- 签到前确认要签到的贴吧
- 手动签到
- 删除已添加的百度账号
- 服务器定时自动签到
- 签到记录和失败原因
- 管理员后台：用户、账号、今日统计、审计日志
- 管理员可删除、封禁用户，并修改普通用户密码
- 遇到验证码 / 风控时暂停账号

## 运行

```bash
python3 -m pip install -r requirements.txt
python3 app.py
```

默认地址：

```txt
http://127.0.0.1:8000
```

可用环境变量：

```bash
HOST=0.0.0.0 PORT=8000 APP_SECRET='change-me' SIGN_DELAY_MIN=0.2 SIGN_DELAY_MAX=0.8 python3 app.py
```

## 数据文件

- `tieba.db`：SQLite 数据库
- `.app_secret`：本地加密密钥，未设置 `APP_SECRET` 时自动生成

生产环境请固定设置 `APP_SECRET`，否则丢失密钥后无法解密已保存的 `BDUSS`。

## 安全边界

- 不保存百度明文密码
- 不绕过验证码
- 不处理异常登录风控
- 账号触发验证码或风控后暂停，等待用户处理
